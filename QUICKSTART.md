## Instant Setup (No Installation Required!)

This AI Trading Assistant runs entirely in your browser - no server, no installation, no build process!

### Test It Right Now

1. **Open `index.html`** in any web browser
2. **That's it!** The app is ready to use

### First Analysis

1. The default stock is **AAPL** (Apple Inc.)
2. Click **"Analyze Stock"** button
3. Wait 3-5 seconds for data to load
4. Review the AI's prediction and reasoning

### Try Different Stocks

Popular examples:
- `AAPL` - Apple
- `MSFT` - Microsoft
- `GOOGL` - Google
- `TSLA` - Tesla
- `NVDA` - NVIDIA
- `AMZN` - Amazon
- `META` - Meta (Facebook)

### What You'll See

1. **Direction Prediction**: BULLISH, BEARISH, or NEUTRAL
2. **Confidence Score**: 0-100% confidence level
3. **Detailed Reasoning**: Why the AI made this prediction
4. **Interactive Chart**: Price history with moving averages
5. **Key Metrics**: Current price, RSI, momentum, volume
